Installation

The program is very simple to install, all you have to do is to run the install script as explained in Installing. The nice part you can use it on other distributions except Decebal Linux too. It is used on Red Hat Linux 7.2, and it does the job it is suposed to do (upgrade the server software, apache, postfix, php and openssh), but you need to have the devel rpm's installed and skip dependencies checks (at least the first time you run it). On Slackware it can be installed and it works out-of-the-box.
