Configuring precompiled update handling

To configure precompiled udpate handling, you have to edit the following file /etc/dpack/update, here is an exemple of this file:

#
# /etc/dpack/update - configuration file for dupdate.
#

# This is the mirror that dupate will use to download packages.
UPDATEMIRROR="packages.decebal.org"

# Thes host will be used to sync the update info files database. These files
# will be installed in $DATABASEDIR/update ($DATABASEDIR is defined in /etc/dpack/common)
RSYNCUPDATERMIRROR="update.decebal.org"
