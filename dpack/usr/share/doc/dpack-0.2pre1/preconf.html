Configuring precompiled package handling

To configure precompiled package handling, you have to edit the following file /etc/dpack/common, here is an exemple of this file:

#
# /etc/dpack/common - common variables used by pkg_add, pkg_del, pkg_info,
#                     dsrc and dupdate.
#

# This is the directory where information about installed packages is saved (the
# package database directory).
DATABASEDIR=/var/lib/pkg

# Directory where pkg_install, dsrc, dsec and dupdate will save downloaded files (there
# will be tree directories in the cache directory, dupdate, dsrc, and dsec)
CACHEDIR=/var/cache

# Directory where pkg_install, pkg_remove, dsrc, dsec and dupdate will log actios.
LOGDIR=/var/log/dpack

# Log method (can be combined, all actions save in one log file, or
# separate, actions are saved in separate files).
LOGMETHOD=combined

# Donwload tool used by pkg_install, dsrc and dupdate to download files (if you
# want a separate tool for dsrc then edit the line in /etc/dpack/build containing
# your download tool, ex: DOWNLOADTOOL="axel"). The program should be able to
# download files from both http and ftp.
DOWNLOADTOOL="wget --continue --passive-ftp --tries=3 --waitretry=3"

# Do you want color messages (usefull for debugging).
USECOLORS=yes

# Do you want sounds for events (not really implmented in this version).
USESOUNDS=no
