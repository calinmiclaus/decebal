History

I used a couple of Linux distributions and one day I decided to create my own Linux distribution and i thought that it would be great if my distro would have its own package manager. So i studied all package managers that came in my hands, Red Hat Package Manager - "rpm", Debian's dpkg and apt-get, portage from Gentoo, pacman from ArchLinux, pkgsrc from NetBSD and pkgtools from Slackware. I was impressed by the simplicity of apt-get to install precompiled .deb packages, Gentoo's portage is great when it comes to sources and Slackware, what to say about it, it cant be more simpe. So i took all that i needed from these tools and wrote the scripts that now are called "dpack". Why did i use bash, first reason, I'm too lasy to write them in C and it's simpler to mantain.
