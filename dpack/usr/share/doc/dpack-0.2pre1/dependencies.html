Handling dependencies

There are two ways of handling dependencies: building dependencies with dsrc or installing them with dupdate. The first one should be used if you want to optimize the packages or if you want to pass specific options to the package (eg: USE="nls" dsrc ed) and the second one will download precompiled packages from the internet and install them.

These options are provided for the user's freedom of choice, i use them only if i dont have time to install each package separately.
