Installing

Download the latest archive dpack-0.2.tar.bz2 and run the following comands as root:

[root@localhost:~]# tar xjf dpack-0.2.tar.bz2
[root@localhost:~]# cd dpack-0.2
[root@localhost:dpack-0.2]# ./configure
[root@localhost:dpack-0.2]# make
[root@localhost:dpack-0.2]# make install
[root@localhost:dpack-0.2]#
