1. Introduction

Decebal Package Manager, also called "dpack", is a simple package manager, that can handle packages from precompiled archives or sources. It is able to install, remove, upgrade and build packages, print information about installed packages, install packages "on-the-fly" from the internet, do automatic security updates for you and "much more". It was designed with simplicity in mind, and it provides all i need to manage packages on my Linux box.
