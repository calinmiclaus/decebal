Requirements

The following packages must be installed (versions are not a must, they are the versions that come with Decebal Linux 0.9.0 rc1, and since i wrote this for this distribution i decided to put them here) before installing dpack:

bash version 3.0, patch level 13
sed version 4.1.4
grep version 2.5.1
gawk version 3.1.4
mktemp version 1.5
gcc version 3.3.4 (with Smash Stack Protecor patch)
wget version 1.9.1
rsync version 2.6.3
tar version 1.15.1
bzip2 version 1.0.2
coreutils version 5.2.1
